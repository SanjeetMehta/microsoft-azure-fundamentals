Azure Notifications Hub:
    Pub/Sub. Send push notifications to any platform from any back end.

Azure API Apps:
    API Gateway. Quickly build and consume APIs in the cloud. Route APIs to Azure Services.

Azure Service Bus:
    Service Bus. Reliable cloud messaging as service (MaaS) and simple hybrid integration.

Azure Stream Analytics:
    Serverless real time analytics, from the cloud to the edge.

Azure Logic Apps:
    Schedule, automate, and orchestrate tasks, business process and workflows. Integration with  Enterprise SaaS and Enterprise applications.

Azure API Management:
    Hybrid, multi-cloud management platform for the APIs across all environments. Put in-front of existing APIs to add additional functionality.

Azure Queue Storage:
    Messaging queue. A data store for queuing and reliably delivering messages b/w applications.
